yiyu gets a facelift! The familiar yiyu interface is powerful but difficult to use on cellphones. We have developed a unified search interface and a species summary page which work well on small screens. Test the new interface and species summary page by clicking on the “New Interface” link on the top right of this webpage, and send us your comments.

In the future, we intend to make the simplified search page the default search page, but don′t worry: the current search page will still be available as the advanced search page.
